## 前言

最近接連看到 AMD RX6000 系列顯卡和 Ryzen 5 3600 在特價，就突然萌生出升級電腦的想法

## 目前電腦規格

目前使用的電腦是我在 2017 年使用部分二手零件組裝的電腦

- 處理器：Intel Core i7 3770（二手）
- 主機板：MSI ZH77A-G43（二手）
- 散熱器：利民 TRUE Spirit 90M Rev.A
- 記憶體：6GB（UMAX DDR3 1333 4G + DDR3 1333 2G）
- 顯示卡：MSI Radeon RX 460 2G OC
- 硬碟：美光 MX300 275GB
- 機殼：銀欣 PS14B
- 電源：台達實在力量 350W

這台是我第一台自己組裝的電腦，因為當時預算不足，所以 CPU 和 MB 選用了二手零件

在手邊沒有多餘零件可以 Debug 的情況下購買二手零組件真的是很魯莽的決定，幸好除了主機板的第四個記憶體插槽有點問題之外就沒有太大的問題了

五年過去，硬體沒變的也只有二手零件的部分，其他零件或多或少都有升級

- 處理器：Intel Core i7 3770（二手）
- 主機板：MSI ZH77A-G43（二手）
- 散熱器：利民 TRUE Spirit 90M Rev.A
- 記憶體：12GB（Kingston DDR3 1333 4GBx2 + UMAX DDR3 1333 4G）
- 顯示卡：MSI Radeon RX 470 ARMOR 4G
- 硬碟：美光 MX300 275GB + 美光 MX500 500GB
- 機殼：銀欣 PS14B
- 電源：海韻 Focus GX-550 550W

## 開箱

雖然開頭有提到顯卡，但其實我想買的 RX 6600 已經賣完了，因次我決定先升級其他部分

那這次升級的規格如下：

- 處理器：AMD Ryzen 5 3600
- 主機板：技嘉 B550M-DS3H
- 記憶體：Kingston FURY Beast DDR4 3200 16GB (8GBx2)

主機板的部分預算只有 2K，原本選定 B550 GAMING X V2 的礦渣

不過要買的時候價格已經回漲了，因此改選另一個礦渣、規格還能接受的 B550M-DS3H ~~（買完之後就發現原賣場偷漲價）~~

![零組件合照](https://i.imgur.com/yqLDflT.jpg)

CPU 是 MPK 版，雖然有附散熱器不過我還是用原本的塔散，畢竟那個水泥膏嘛...

![AMD Ryzen 5 3600 正面特寫](https://i.imgur.com/CJS02cM.jpg)

從包裝上的貼紙就能知道這張主機板一定是礦渣，但至少是全新的

![技嘉 B550M-DS3H 包裝正面](https://i.imgur.com/PNGVdjw.jpg)

技嘉的 B550M-DS3H 在這個價位算比較好的那個，有 5 + 3 項電源和四組記憶體插槽

其他價位相似的不是只有 4 + 2 項電源，就是只有兩組記憶體插槽

![技嘉 B550M-DS3H 正面](https://i.imgur.com/KAo5jop.jpg)

記憶體就不多說了，以後有機會再插滿

![Kingston 記憶體包裝特寫](https://i.imgur.com/b59WcvO.jpg)

組裝完就是這個樣子（整線已經放棄，線不要折到就好）

![電腦組裝完畢後的內部圖](https://i.imgur.com/47clnG7.jpg)
